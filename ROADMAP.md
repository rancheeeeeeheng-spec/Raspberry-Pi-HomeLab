# Roadmap

This roadmap outlines the planned evolution of the Raspberry Pi HomeLab. Each milestone builds upon the previous one, introducing new technologies while reinforcing practical skills in Linux, networking, containerization, automation, and infrastructure management.

---

# Phase 1 – Foundation ✅

Establish the core infrastructure required to support self-hosted services.

### Objectives

- [x] Install Raspberry Pi OS Lite
- [x] Configure SSH for remote administration
- [x] Update and secure the operating system
- [x] Install Docker
- [x] Install Docker Compose
- [x] Deploy Portainer
- [x] Deploy Homepage
- [x] Deploy Pi-hole
- [x] Deploy Uptime Kuma
- [x] Create GitHub repository
- [x] Write technical documentation

**Status:** Completed
